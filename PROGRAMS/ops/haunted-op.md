# HAUNTED — Ghost Detector (Operative)

> **Source theory**: haunted.json  
> **Input**: Any .md file in the worldtext  
> **Output**: A ghost/machinery report

## What It Does

HAUNTED is Calvino's ghost-finder. It reads a file and asks: **does this text produce more than its structure explains?** The machinery is the template, the format, the genre. The ghost is the specific sentence that could not have been predicted from the template alone.

## Operations

### 1. TEMPLATE_EXTRACTION
Identify the structural template of the file:
- Does it follow a standard format (## Ontology, ## Places, ## Key Entities, etc.)?
- How many section headers match the world-page template?
- What percentage of the file is template (headers + boilerplate) vs. content (specific claims)?

```bash
# Count template vs. content
TARGET="worldtext/worlds/world-thick-description.md"
template_lines=$(grep -c "^## \|^> \*\*\|^---\|^- \[\[" "$TARGET")
total_lines=$(wc -l < "$TARGET")
content_ratio=$(echo "scale=2; ($total_lines - $template_lines) / $total_lines" | bc)
echo "Content ratio: $content_ratio (< 0.5 = template-dominated, > 0.7 = content-rich)"
```

### 2. GHOST_SCAN
Search for sentences that break the template — claims, images, or formulations that could NOT have been generated by filling in a generic world-page form:

**Ghost indicators**:
- Sentences > 30 words with no wiki-links (narrative voice, not reference voice)
- Metaphors that don't use the standard vocabulary (Rylean, Geertzian, operative, etc.)
- Quotes or direct speech
- Self-contradiction or acknowledged tension
- Humor, regret, or confusion

**Machinery indicators**:
- Sentences that begin "This world..." or "The key entities are..."
- Bullet lists of wiki-links
- Standard section headers
- Sentences that could be copy-pasted to any other world page with only the proper nouns changed

### 3. THE CALVINO TEST
Extract the 3 most distinctive sentences in the file — the ones with the highest "ghost density" (specific, surprising, untemplate-able). Also extract the 3 most generic sentences (pure machinery).

Report them side by side. The operator decides: is the ghost real, or is it just unfamiliar machinery?

### 4. SURPLUS_DETECTION
Compare the file against the source paper that produced it. Find sentences in the synthesis/world that have NO equivalent in the source paper. These are the surplus — generated by the combinatorial intersection of the source with the worldtext's existing vocabulary.

```bash
# Find surplus: sentences in world that don't match source
WORLD="worldtext/worlds/world-thick-description.md"
SOURCE="PAPERS/geertz.md"
# Extract one-sentence claims from world
grep -oh '[A-Z][^.]*\.' "$WORLD" | while read sentence; do
  # Check 3-word windows against source
  key=$(echo "$sentence" | awk '{print $1, $2, $3}')
  if ! grep -q "$key" "$SOURCE" 2>/dev/null; then
    echo "SURPLUS: $sentence"
  fi
done
```

## Report Format

```
HAUNTED REPORT: world-thick-description.md
========================================
Content ratio: 0.72 (content-rich)
Template lines: 28
Content lines: 71
GHOSTS (3 best):
  1. "Its atmosphere is closer to a therapy session than a writing workshop"
  2. "The machine was never at this altitude"
  3. "Culture is what happens when the thin version fails"
MACHINERY (3 worst):
  1. "This world extends far beyond ethnography"
  2. "Key entities include..."
  3. "The core distinction is..."
SURPLUS: 12 sentences with no source equivalent
VERDICT: MODERATELY HAUNTED — file has genuine ghosts in its atmosphere sections but defaults to machinery in its cross-links
```

## Definition

The **spatial operator** does not merely arrange objects in space. It **writes encounter geometry** — the spatial shape of an encounter as experienced by a body, tool, or agent moving through it.

## Core Operations

| Operation | Effect |
|-----------|--------|
| **Positions** | Places body, object, and interface relative to each other |
| **Modulates distance** | Not just meters but *traversal cost* and *felt proximity* |
| **Modulates orientation** | Toward, away, alongside, around |
| **Modulates access** | Opens paths, closes paths, filters approach |
| **Thickens thresholds** | Makes crossings *cost* something — attention, effort, permission |
| **Bends approach** | Routes movement through detour, delay, or indirection |
| **Folds elsewhere into here** | Collapses remote relations into local encounter |

## Ecological Syntax

The spatial operator generates what the theory calls **ecological syntax**: the condition where space begins to instruct relation.

```
<Ecological Syntax> [binds] <Posture> [to] <Inference>
<Ecological Syntax> [binds] <Distance> [to] <Permission>
<Ecological Syntax> [binds] <Orientation> [to] <Expectation>
<Ecological Syntax> [binds] <Threshold> [to] <Method>
```

Ecological syntax is not language *about* space. It is space *when space begins to instruct relation*.

## Work on Prepositions

The spatial operator does its deepest work on prepositions:

| Preposition | What the Operator Makes of It |
|-------------|------------------------------|
| *near* | Measured hesitation |
| *through* | Conditional passage |
| *across* | Vector of attention |
| *behind* | Operational blindness |
| *between* | Negotiated interval |
| *toward* | Motor commitment |

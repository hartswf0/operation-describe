## Definition

**Text is not a representation. Text is a control signal.**

In cybernetic terms (Wiener 1948), text is a governor — a regulatory mechanism that steers a receiver's sensory processing toward a target state. It does not "describe" an image; it **causes** the receiver (human or AI) to generate one.

This reframing dissolves the word/image problem (paragone) that has structured ekphrasis theory since Lessing. There is no "translation" between modalities because text was never in a separate modality to begin with. Text is:

1. **Acoustic** (phonemes are sound waves)
2. **Motor** (articulation requires body movement; typing requires touch)
3. **Visual** (graphemes are spatial patterns on surfaces)
4. **Temporal** (language unfolds sequentially)
5. **Semantic** (words bear conceptual content)

Text is ALREADY multimodal. The "word/image gap" is a semiotic abstraction imposed on a continuous cross-modal signal.

## The Five Functions of Text

| Function | Mechanism | Ancient Term | Modern Term |
|----------|-----------|-------------|-------------|
| **Transduce** | Convert energy between sensory modalities | Mimesis (imitation through different medium) | Encoder-decoder architecture |
| **Compress** | Reduce high-dimensional experience to sequential signal | Brevitas (condensed expression) | Tokenization / latent encoding |
| **Trigger** | Activate cross-modal gestalts via sound symbolism | Enargeia ("bringing before the eyes") | CLIP cross-modal alignment |
| **Steer** | Modulate receiver toward target perceptual state | Persuasio (rhetorical control) | Classifier-free guidance |
| **Leak** | Exploit incomplete sensory separation | Synaesthesis (union of senses) | Embedding space geometry |

## Implications for the Cosmos

If text is a control signal rather than a representation, then:

- **Ekphrasis** is not "verbal representation of visual representation" but **verbal steering of visual generation**
- **Prompt engineering** is not "describing desired output" but **tuning the control signal for maximum cross-modal activation**
- **Thick description** (Geertz) is not "rich representation" but **maximizing the bandwidth of the control signal** — encoding more cross-modal information per word
- **Enargeia** is not "vivid description" but **high-gain cross-modal triggering** — words that activate visual, motor, and emotional processing simultaneously
- **The pictorial third** (Louvel) is not a mysterious emergence but the **constructive interference pattern** between the control signal (text) and the receiver's generative system (phantasia / diffusion model)

## The Hierarchy of Text Quality

This framework predicts a hierarchy of text quality based on cross-modal bandwidth:

| Level | Text Type | Cross-Modal Bandwidth | Example |
|-------|-----------|----------------------|---------|
| 5 | Enargetic ekphrasis | Maximum: sound, rhythm, semantics, motor, affect all activated | Homer's Shield of Achilles |
| 4 | Thick description | High: semantics + contextual + cultural + sensory | Geertz's Balinese cockfight |
| 3 | Descriptive prose | Medium: semantics + some sensory | Standard art criticism |
| 2 | Technical specification | Low: semantics only, single modality | "Generate a 512x512 image of a cat" |
| 1 | Label | Minimal: one semantic token | "cat" |

The best prompts operate at Level 4-5. The worst prompts operate at Level 1-2. **Prompt engineering is the art of climbing this hierarchy** — adding cross-modal bandwidth to text until the control signal activates the same perceptual gestalt the user intends.

This is exactly what the progymnasmata curriculum taught: how to climb from Level 1 (naming) to Level 5 (enargeia). The same curriculum applies to prompt engineering.

# Conflict: Description vs. Generation

> **Species**: conflict_grammar  
> **Scale**: fragment  
> **Primary World**: [[world-operative-ekphrasis]]  
> **Last Revision**: 2026-04-13  

---

The collapse of the distinction between describing and producing. Bajohr (2024) argues this collapse is technically real in multimodal AI. But does a residue of the old distinction persist in human experience? The prompter still *intends* description even as the machine *performs* generation. This tension generates the world of operative ekphrasis.

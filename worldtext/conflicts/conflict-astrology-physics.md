## The Conflict

Geertz draws the distinction between astrology and physics to illustrate the difference between thin universals and thick particulars. Astrology applies the same framework to every person born under the same sign — the universal swallows the particular. Physics applies universal laws but measures specific systems — the universal serves the particular.

The conflict in this cosmos: are the 19 program theories physics (instruments that measure specific texts and produce specific readings) or astrology (frameworks that apply the same categories to everything and produce the same conclusions)?

The test: if you run `haunted.json` against a grocery list instead of Murdoch, does it still find a "ghost"? If yes, it's astrology. If no — if it returns "no ghost detected, this text is not haunted" — it's physics. The program is only real if it can produce a negative result.

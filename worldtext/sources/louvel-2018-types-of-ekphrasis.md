## Citation

Louvel, Liliane. "Types of Ekphrasis: An Attempt at Classification." *Poetics Today* 39, no. 2 (2018): 245–263.

## Core Argument

Louvel builds the most comprehensive typology of ekphrastic functions in the literature, then proposes a phenomenological concept — the **"pictorial third"** — to account for what happens in the reader's mind/body when word and image interact.

### The Typology

| Type | Function | Example |
|------|----------|---------|
| **Narrativizing** | Disguises narration as description (the Shield) | Homer, Krieger |
| **Hermeneutic** | Functions as clue; reader becomes detective | Banville's *Ghosts* |
| **Maieutic** | Triggers emergence of idea/feeling in character | Wharton's *House of Mirth* |
| **Subversive / Debunking** | Contradicts or undermines the image described | Barnes's *History of the World* |
| **Anachronistic** | Juxtaposes past visual conventions with present values | Hemon's *Lazarus Project* |
| **Monumental / Elegiac** | Commemorates; erects verbal monument | Findley's *The Wars* |
| **Affective** | Expresses longing to merge with the image (Narcissus/Pygmalion) | James's *Ambassadors* |
| **Pragmatic / Technical** | Describes artist's technical choices and process | Spencer's notebook |
| **Creative** | Describes the advent/eruption of creative inspiration | Spencer's letters |
| **Critical** | Art criticism as second-order ekphrasis | Hustvedt's *Mysteries of the Rectangle* |
| **Heuristic** | Links text with discovery; ekphrasis as knowledge production | Spencer's Oxford lectures |

### The Pictorial Third

Louvel's most important theoretical contribution: when ekphrasis activates the reader's imagination, something arises **between** the text and the image — neither purely verbal nor purely visual. This "pictorial third" is:

> "a 'relation,' a dynamic activity... a modality sharing qualities with the living, a lived experience, an 'event' in the meaning of what happens: an intermedial performance." (p. 261)

The pictorial third is **not a blend** (in the Fauconnier/Turner sense) but an emergence — something new that neither text nor image alone contains.

## THE BRIDGE: Louvel's Typology Maps onto Prompt Engineering

This is the connection no one has made. Louvel's 11 types of ekphrasis map directly onto 11 modes of AI prompting:

| Ekphrastic Type (Louvel) | Prompt Engineering Equivalent |
|-------------------------|-------------------------------|
| **Narrativizing** | Narrative prompts ("a knight entering a forest, approaching a castle...") |
| **Hermeneutic** | Interrogative prompts ("what would it look like if...") |
| **Maieutic** | Exploratory prompts (underdetermined, seeking surprise) |
| **Subversive** | Adversarial / jailbreak prompts (subverting the model's training) |
| **Anachronistic** | Style-mixing prompts ("medieval painting of a Tesla factory") |
| **Elegiac** | Memorial / nostalgic prompts ("faded photograph of...") |
| **Affective** | Atmospheric prompts ("dreamy, ethereal, longing...") |
| **Technical** | Specification prompts ("8K, ray-traced, 50mm lens, f/2.8") |
| **Creative** | Serendipity prompts (random combination seeking emergence) |
| **Critical** | Evaluation prompts (assessing generated output) |
| **Heuristic** | Research prompts (using generation to discover visual possibilities) |

### The Pictorial Third in AI

When a user writes a prompt and receives a generated image, the "pictorial third" appears in the gap between what they wrote and what they see. The generated image is never exactly what the prompt described — it is an *emergence* from the interaction of text and model. The user's experience of this gap (surprise, recognition, disappointment, wonder) IS the pictorial third of operative ekphrasis.

**This means the pictorial third is not limited to human readers.** It is a structural feature of any system where verbal input produces visual output through a mediating process (cultural memory / neural network / the reader's imagination). The pictorial third is the **gap itself** — the distance between prompt and image, between description and visualization, between word and vision.

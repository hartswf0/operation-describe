## The Flow

An object (a description, an image, a prompt, a sheep) passes through multiple interpretive frames. At each threshold, its identity changes:

```
Object-in-Frame₁ → [threshold] → Object-in-Frame₂ → [threshold] → Object-in-Frame₃ → ...
```

### Cohen's Sheep
500 sheep → (mezrag frame) = legitimate indemnity → (French colonial frame) = contraband → (captain's equivocation) = pragmatic ambiguity → (colonel's jurisdiction) = theft

### W's Cover Image
AI-generated cover art → (design studio) = client-approved creative solution → (hostile reviewer) = "AI slop," evidence of creative bankruptcy → (portfolio) = case study → (art gallery) = statement about process

### A prompt's passage
"A carrot civilization" → (prompter's context) = architectural experiment → (LDraw compiler) = geometric instruction set → (LEGO Community) = possible homage or appropriation → (art critic) = commentary on automation → (training data of future model) = one more data point in the distribution

## What Flows

The object itself does not change. What flows is **meaning** — carried by the object as it crosses thresholds between interpretive communities. Each community applies its own norms (Fish), fills the object's gaps differently (Iser), and produces a different concretization (inference).

**Thickness** is the degree to which the original description/prompt anticipated these reclassifications. A thin prompt survives only one frame. A thick prompt carries enough contextual density to remain legible across multiple frames. Geertz's Layer 6 (reclassification) is the engineering of durability across thresholds.

# Epoch: Thick Description Diffusion (1973 – 2020)

> **Species**: epoch  
> **Scale**: system  
> **Period**: 1973 – 2020  
> **Locale**: Anthropology → humanities → design → AI  
> **Last Revision**: 2026-04-13  

---

Geertz (1973) articulates thick description as interpretive method. Over 50 years, the concept diffuses across disciplines: literary criticism, philosophy, design research, HCI, and eventually AI evaluation ("thick evaluation"). Each crossing deposits a new layer without erasing older ones.

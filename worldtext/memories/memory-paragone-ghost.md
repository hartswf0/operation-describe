## The Trace

The paragone — the ancient competition between word and image for representational supremacy — is structurally dead. Bajohr's operative ekphrasis collapses the distinction: text and image are the same data type in multimodal AI. But the paragone **persists as a ghost** in the cosmos. Scholars still invoke it. Critics still frame AI imagery as "word vs. image." The distinction survives its own dissolution.

## Why It Persists

1. **Institutional inertia**: Academic departments are organized around the paragone (literature departments, art history departments). The collapse has no institutional home.
2. **Phenomenological residue**: The human user still *experiences* the prompt as text and the output as image. The computational collapse is invisible to the senses.
3. **The new paragone has no name yet**: "Intent vs. distribution" has replaced "word vs. image," but the new conflict hasn't been theorized enough to dislodge the old one from scholarly habit.

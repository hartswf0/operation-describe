## The Five Stages

| Stage | Function |
|-------|----------|
| **DELIMIT** | Establish the diegetic boundary and governing physical rules |
| **SEGMENT** | Divide the source into macro units, beats, syntagmas, and atomic patches |
| **TRACK** | Record measurable change across bodies, objects, camera, light, sound, and time |
| **COMPRESS** | Convert each patch into a concise generative seed and invariant state description |
| **TRANSMIT** | Output reconstruction matrices for human analysis and machine conform |

## Segmentation Protocol

| Level | Definition |
|-------|-----------|
| **Diegesis** | Total physical boundary of the source world: entities, locations, rules, invariants |
| **Macro Units** | Structural blocks defined by severe shifts in space, time, information load, or action regime |
| **Beats** | Causal shifts: action reversal, threshold breach, force redistribution, entry/exit, state change |
| **Syntagmas** | Editing law governing a beat: continuous, alternating, parallel, interrupted, bracketed, accumulative, composite |
| **Patches** | Atomic local event window. Target: 8–10 seconds. Hard limits: 6–12 seconds |

---

## Patch Trigger Rules

Slice a new patch immediately when:
- Location coordinates shift
- Primary body executes a new kinetic vector
- Camera changes optical distance
- Camera changes angle, axis, or movement regime
- Ambient light state changes
- Dominant sound regime changes
- Time-arrow jumps
- Entity enters or exits, changing active geometry
- Object crosses threshold, redistributing attention or force
- New subsystem becomes dominant in frame

Patch boundaries obey physical event coherence, not convenience.

## Analytic Kernel

For each level of extraction, identify the source as a system:

**Measurable Properties**:
- Variables (changing quantities)
- Inputs (initiating conditions)
- Outputs (resulting visible/audible states)
- States (current operative conditions)
- Constants (stable recurring values)
- Invariants (relations persisting across transformation)

**Structural Components**:
- Axioms (accepted operating truths)
- Primitives (irreducible entities or actions)
- Postulates (necessary reconstruction assumptions)

**Property Classification**:
- Fundamental or Derived
- Scalar or Vector

**Dynamic Tracking**:
- Feedback loops (reinforcing or balancing)
- Equilibrium tendencies (attractors or repellers)
- System type (open or closed)
- Subsystems (dependent or independent)
- Constraints (hard or soft)

## Observation Discipline

**BANNED**: abstract mental verbs (feels, thinks, realizes, knows, understands, wants), symbolic inflation, theme invention, plot hallucination, unsupported causal claims, placeholder metadata.

**ALLOWED ONLY**: externally visible action, audible change, measurable relation, defensible inference from repeated physical evidence.

**Replace**:
- NOT: "she fears him"
- BUT: "her shoulders pull back, gaze snaps left, feet retreat two steps"

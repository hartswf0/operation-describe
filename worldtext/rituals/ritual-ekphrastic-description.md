# Ritual: Ekphrastic Description

> **Species**: ritual  
> **Scale**: fragment  
> **Primary World**: [[world-classical-ekphrasis]]  
> **Last Revision**: 2026-04-13  

---

The foundational ritual: rendering a visual object present to a reader/listener through language. Stabilizes the world of classical ekphrasis by perpetuating the word-image encounter across centuries. From Homer's oral performance to contemporary ekphrastic poetry, the ritual maintains the same structure: a speaking voice confronts a silent image and attempts to translate its visual power into verbal form.

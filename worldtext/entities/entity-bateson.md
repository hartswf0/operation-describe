## Identity

Gregory Bateson (1904–1980). Anthropologist, cyberneticist, systems theorist. Author of *Steps to an Ecology of Mind* (1972) and *Mind and Nature* (1979). Husband of Margaret Mead. One of the founding theorists of cybernetics alongside Wiener and Von Foerster.

## Key Contributions to the Cosmos

### 1. Learning Levels (I / II / III)
- **Learning I**: Stimulus-response. A prompt producing an output.
- **Learning II**: Learning *about* the context of stimulus-response. Learning the rules of the game. Persona, tone, cultural code.
- **Learning III**: Learning *about* the paradigm that structures Learning II. The meta-context. "This is speculative fiction" / "Truth constraint: relaxed."

**Cosmos application**: The three levels map onto prompt stratigraphy. Mixing injunctions across levels (e.g., "be creative" at Level II + "follow rules exactly" at Level II) produces a **double bind** — contradictory constraints that cause mode collapse. The XML container strategy proposed in "Second-Order Thick Prompting" enforces logical type separation.

### 2. The Double Bind
A double bind occurs when a subject receives contradictory injunctions at different logical levels, with no ability to metacommunicate about the contradiction. Bateson proposed it as the communicative structure underlying schizophrenia.

**Cosmos application**: Prompts that contain contradictory constraints at different logical levels produce AI analogues of the double bind: incoherent outputs, mode collapse, bizarre compromise generations. The thick prompt must not only specify layers but **separate** them by logical type.

### 3. "A Difference That Makes a Difference"
Bateson's definition of information: not a thing but a relationship — a difference in the world that produces a difference in the receiver.

**Cosmos application**: A prompt token is information only if it changes the probability distribution of the output. A token that has no effect on the generation is noise, not information. This provides a **formal test** for prompt efficiency: each token should make a measurable difference (the Δ operator from the GOT papers).

### 4. Pleroma / Creatura
Bateson borrowed Jung's terms: **Pleroma** (the non-living world, governed by forces) and **Creatura** (the living world, governed by differences/information). AI occupies the boundary: it is Pleroma (silicon, electricity) that simulates Creatura (meaning, narrative, intention).

**Cosmos application**: The "Carrot Uprising" — an AI-generated narrative about vegetables gaining sentience — enacts this boundary literally. The Pleroma (computation) generates a story about the Pleroma (vegetables) becoming Creatura (sentient beings). Second-order simulation of life.

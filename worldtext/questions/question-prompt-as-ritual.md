# Question: Is Prompting a New Ekphrastic Ritual?

> **Species**: question  
> **Scale**: world  
> **Frontier**: Does prompting stabilize a new world in the same way ekphrastic description stabilized the classical world?  
> **Filed**: 2026-04-13  

---

Classical ekphrastic description is a ritual: a repeated, stabilizing act that maintains the word-image encounter. Prompting shares structural features — repeated, iterative, language-mediated engagement with visual production. But it differs in that the visual output is *generated*, not pre-existing.

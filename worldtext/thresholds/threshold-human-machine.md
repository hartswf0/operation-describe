# Threshold: Human / Machine

> **Species**: threshold  
> **Scale**: fragment  
> **Primary World**: [[world-prompt-craft]]  
> **Last Revision**: 2026-04-13  

---

The boundary where human intent meets algorithmic generation. The prompter operates on one side (natural language, intention, aesthetic judgment); the model operates on the other (embeddings, attention, sampling). The prompt interface is this threshold's physical location.

# Distinction: actual / notional

> **Primary World**: [[world-classical-ekphrasis]]  
> **Last Revision**: 2026-04-13  

---

**Poles**: actual / notional

Describing a real artwork (actual) vs. an imagined one (notional). Homer's Shield is notional — it exists only in the text. This distinction questions whether ekphrasis requires a pre-existing visual referent.

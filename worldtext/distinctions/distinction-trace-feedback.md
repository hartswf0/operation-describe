# Trace / Feedback

> **Species**: distinction  
> **Poles**: trace logic / feedback logic  
> **Parent world**: [[world-operative-ethnography]]

---

## The Distinction

| | Trace Logic | Feedback Logic |
|---|-----------|---------------|
| **Flow** | World → Encounter → Description | World → Encounter → Description → ↻ Encounter |
| **Description** | Follows the event | Re-enters the event |
| **Temporality** | Linear, retrospective | Recursive, prospective |
| **Effect** | Archive | Intervention |
| **Observer** | Participant observer | Operative observer |

## Trace Logic

In trace logic, description is a **secondary artifact**. The world produces an encounter, the encounter produces a trace, the trace becomes a description. The description looks backward.

```
<World> [produces] <Encounter>
<Encounter> [produces] <Description>
```

## Feedback Logic

In feedback logic, description is an **operational recirculation**. The description re-enters the encounter conditions and alters what can happen next. The description looks forward.

```
<World> [produces] <Encounter>
<Encounter> [produces] <Description>
<Description> [reprograms] <Encounter Conditions>
<Encounter Conditions> [produce] <New World>
```

## Why It Matters

This distinction is the epistemological engine beneath the [[distinction-participant-operative]] distinction. Participant observation operates in trace logic: the ethnographer writes *after* the encounter. Operative ethnography operates in feedback logic: the description *is* part of the field.

This is also why [[concept-worldtext|worldtext]] is not merely a report. It is a **description that re-enters the world as a new condition for future emergence**.

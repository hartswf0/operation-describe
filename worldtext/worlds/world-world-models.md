## Ontology

Ha & Schmidhuber (2018) demonstrated that reinforcement learning agents can learn a compressed **world model** — a generative neural network that simulates the environment. Then the agent can be trained *entirely inside its own hallucinated dream*. The dream-trained policy can transfer back to the actual environment.

This is ekphrasis inverted. The poet describes a world that already exists. The world model *generates* a world that does not yet exist — and then inhabits it. The dream is the place; the agent is the ekphrast and the audience simultaneously.

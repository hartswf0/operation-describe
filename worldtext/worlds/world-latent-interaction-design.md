## Thesis

HCI is trying to govern a wetland with the tools of a hallway.

The field already knows that generative models are unstable, opaque, and socially consequential. The anomaly is that it still studies them as if they were stable interfaces. That mismatch is no longer a minor methodological issue. It is a category error producing false knowledge at exactly the moment when human-model relations are becoming infrastructural.

**Latent Interaction Design** is a practice-centered research program that studies prompting as embodied, relational steering through a changing latent field, using artifacts, trajectories, context windows, harnesses, and longitudinal contact as primary evidence.

## The Sixteen Operating Rules

### Evidence Rules

1. **Enter the Model.** Open the model. Set a target. Run the prompt. Record the output. Change one variable. Run it again.
2. **Contact Logs, Not Interviews.** Begin with contact logs: prompt, settings, output, correction, outcome, timestamp, model version.
3. **The Model Is a Changing Response Surface.** Re-run old prompts against new model states. Compare the difference.

### Ontological Rules

4. **Prompting Is Control, Not Command.** A prompt is a control move inside a probabilistic system (Wittgenstein: "the meaning of a word is its use").
5. **Navigation, Not Usability.** Can the practitioner detect resistance? Correct course? Reach the target under drift?
6. **Working Paths, Not Best Prompts.** A working path = target + sequence + failures + corrections + breaking conditions. Call this a **Promptline**.

### Methodological Rules

7. **Traces, Not Summaries.** Put novices and experts on the same model with the same target. Record every move. Compare traces.
8. **Apprenticeship, Not Extraction.** Co-prompt beside skilled practitioners with the screen live. Do not reduce to retrospective testimony.
9. **Failure Is Data.** Log, classify, test: framing conflict? style prior? safety intervention? context loss? model drift?

### Design Rules

10. **Expose the Variables.** A person cannot steer what the interface conceals: model version, system prompt class, context window pressure, temperature.
11. **Live Instruments, Not Fixed Heuristics.** Show output variation, failure clusters, version differences, edit impact, hard-to-reach regions.
12. **Hydrology, Not Guardrails.** Shape flows: change defaults, reward correction, surface uncertainty, mark provenance, expose system state.

### Relational Rules

13. **Measure Relationality Through Duration.** Track: when does the person stop checking? Over-trust? Hand off judgment? Re-enter the loop?
14. **The Body Shows Up in Rhythm.** Count attempts, reversals, reformulations, mode-switches (direct instruction → analogy → negative constraint).
15. **Enter Before You Moralize.** Test interventions under actual use. Show traces of harm-reduction, utility-destruction, and misuse-displacement.
16. **Support Disciplined Trial, Not Fantasy.** Provide history, branching, diffing, annotation, replay, and controlled comparison.

## The Three Rituals

### [[ritual-navigation-challenge]]
Set a target the model resists. Run three modes: **shortest path**, **longest path**, **strangest path**. Score by target fidelity, route clarity, and documented corrections. This is mud cartography.

### [[ritual-prompt-battle]]
Two practitioners, same model, same target, limited time. Record all moves. Judge output quality and route quality separately. This is deep play (Geertz, 1973). The cockfight of the latent space.

### [[ritual-promptlines]]
Compose a reusable traversal: prompt sequence, output landmarks, failure points, correction rules. Store as living routes. Retest after model updates. This is the Homeric formula for the latent age.

## The New Unit of Analysis

Not the interview. Not the opinion survey. Not the static benchmark.

**The encounter trace**: a timed sequence of prompt, output, correction, comparison, and outcome.

The encounter trace IS the operative ethnography loop ([[world-operative-ethnography]]) formalized as research instrument:

```
configure → encounter → trace → describe → recirculate
```

## The Compressed Theses

**Academic**: Generative AI has exposed a methodological contradiction in HCI: the field describes model interaction as unstable and emergent, yet continues to study it with techniques built for stable interfaces and legible tasks.

**Polemical**: HCI is trying to govern a wetland with the tools of a hallway.

**Journal-ready**: The central failure of current HCI work on generative AI is not ethical insufficiency but methodological misfit: a volatile cultural medium is still being studied as though it were a predictable interface.

**Manifesto**: The latent space will not submit to the interview, the sticky note, or the usability score; it must be entered, endured, and learned in contact.

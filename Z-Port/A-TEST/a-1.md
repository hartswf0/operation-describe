# The Ekphrastic Object: From Image Output to World Model

**Abstract / Core Thesis**
The old ekphrastic question asked whether words could make readers see. Generative AI appears to answer yes: the word becomes the picture. However, that answer is already obsolete. The decisive shift in contemporary computational media is not from word to image, but from word to *world model*. Prompting is becoming a cumulative, procedural, and increasingly spatial practice in which description no longer points at a world from the outside; description helps constitute the world’s operating conditions. That is why Homer’s Shield of Achilles matters. The shield is not merely the first great ekphrasis; it is a miniature world model. It is therefore the hinge between operative ekphrasis and *worldtext*: a described artifact that already behaves like a world-bearing, state-tracking system.

---

### Introduction: The Invalidation

There is a dead coping mechanism in contemporary media theory: pretending that ekphrasis is still mainly a poem describing a picture, or that AI image generation is only a tool for producing finished images.

For centuries, the word reached toward the image. The word named the image. The word described the image. The word tried to make the image appear before the listener’s eye. That effort was never small. It shaped classical rhetoric. It shaped poetry. It shaped the long dream that language could cross the border between saying and seeing. Ekphrasis was that border-work: the old discipline of making the word press hard enough against the image that the reader almost saw.

Then the machine arrived, and the old border did not vanish. The old border became operational.

The prompt is still a word. The output is still an image. But the relation between word and image has ruptured. The word no longer waits inside the poem, asking the reader to imagine. The word enters the system. The word activates the system. The word generates the image. That is the rupture. Not because computers became creative, or because machines became artists, but because the ancient wish of ekphrasis has become a procedure.

The word now does work. The word moves through captions, metadata, scraped images, weighted matrices, diffusion steps, latent spaces, interface defaults, system prompts, and user iterations. The word does not stand outside the image and point. The word is already inside the image. The image is already carrying the mathematical trace of the word.

This is why our old critical vocabulary starts to fail. The image is not a natural sign. The image is not pure vision. The image is not the thing itself finally rescued from language. The image is an image of images. It is a statistical memory of images—a platform’s learned compromise between the prompt, the archive, the model, and the visual habits of a culture already trained to recognize itself.

We cannot save ekphrasis by pretending the machine finally gives us the natural sign. We cannot save ekphrasis by pretending the prompt merely describes. We cannot save ekphrasis by pretending the final image is the point. The final image is not the point. The process is the point.

### I. Prompting as Performance and Thick Description

Prompting is often mistaken for description because it begins as language. That mistake keeps the analysis trapped at the wrong scale. A prompt does not simply say what an image should contain. A prompt tests what a model can make available, what a platform will default toward, what a system will resist, and what a user must restate, thicken, or redirect.

The operative unit of generative media is therefore not the single prompt. The operative unit is the loop.

A user writes a prompt, receives an output, reads the system’s misinterpretation, revises the prompt, adds constraints, changes style, supplies reference images, removes noise, and tries again. The image is not the end of this process. The image is diagnostic feedback. It tells the user what the model heard, what the model ignored, and what the model already assumes.

To substantiate this performative loop empirically, consider a documented interaction with a generative platform such as Midjourney v6. A user inputs a thin prompt: `"A Homeric shield."` The system does not return a neutral historical artifact; guided by its probabilistic training distribution, it defaults to heavily weighted video-game concept art styles. The user reads this misinterpretation and iterates a *thick prompt*, applying negative constraints to sculpt the latent space (`--no unreal engine, fantasy, glowing`) alongside positive structural logic (`concentric rings, agricultural labor, ocean rim, style raw`). The first image was not a failure; it was evidence revealing the model's biases. The prompt is performed against the system. The output answers back. The next prompt incorporates that answer.

This iterative resistance is why *thick prompting* matters. Following Geertz (1973), we must actively deploy thick description as a rigorous methodology rather than a decorative analogy. Geertz insists that a physical action (a wink) cannot be understood outside the stratified hierarchy of cultural codes that give it meaning. Prompting must be analyzed in the exact same way. It is a semiotic negotiation with the accumulated cultural codes embedded in the latent space.

A thin prompt enters a thick system. The result is never a pure translation from intention to image; it is a negotiation with accumulated visual culture. Thick prompting therefore does not mean writing longer prompts—a long prompt can still be fundamentally "thin" if it merely piles up adjectives. It means building a more explicit operating context. The user supplies not only subject matter but relations, constraints, exclusions, styles, references, temporal cues, spatial cues, and procedural instructions.

In this sense, prompting moves closer to performance than to description. Operative ekphrasis makes description executable, and thick prompting extends that executability into an iterative practice. The prompt becomes less like a caption and more like a score: a set of instructions that can be replayed, revised, interpreted, and varied by a system. There is no final perfect prompt because there is no final exhaustion of the model.

### II. The Shield as Worldtext

To understand where this performative iteration leads, we must look backward. Homer’s Shield of Achilles should not enter this argument merely as a famous example. It should enter as the architectural model.

The shield does not depict scenes. It organizes a world. Homer places cities, fields, vineyards, herds, dances, trials, armies, stars, and Ocean on a single made object. The result is not a picture inside the *Iliad*, but a compressed world-system inside the *Iliad*. The shield belongs to the narrative, yet it exceeds the narrative. It appears inside a war poem, but it contains more than war. It holds the conditions that make the Homeric world intelligible.

That is why the shield already behaves like a *worldtext*. It is verbal, visual, spatial, temporal, and procedural at once. Homer does not simply describe a finished object. He describes Hephaestus making it. Scene follows scene. Region joins region. The shield accumulates a world through the act of fabrication. The ekphrasis is therefore not only representational; it is constructive.

This matters for generative AI because the central object of prompting is beginning to move beyond the generated image. In image models, the prompt generates a bounded visual artifact. In world models, the prompt points toward an environment: a space with continuity, affordances, physics, memory, and possible action.

This marks the passage from imagetext to worldtext. An imagetext binds word and image (Mitchell, 1994). A worldtext binds word, image, space, action, and possible life. The shield does not merely ask the listener to see a picture. It asks the listener to enter an ordered totality. The scenes on the shield imply movement and time: the wedding procession moves, the dancers turn, the farmers cut, the armies clash.

A computational world model attempts to do the exact same thing. For formal definitions of algorithmic spatial-temporal state representations, see Ha & Schmidhuber (2018). In contemporary AI, a world model does not merely represent isolated objects; it encodes relations among objects, actions, spaces, temporal sequences, and possible changes. The movement from image model to world model is therefore not merely a move from stillness to motion. It is a move from depiction to state.

*Worldtext* names this shift. It does not replace operative ekphrasis; it extends it. A worldtext is not simply a text about a world. It is a text-image-procedure assemblage that helps bring a world-model into operation. The word no longer aspires only to become a picture. The word begins to specify the conditions under which a world can appear and remain manipulable.

### III. Conclusion: The Door in the Floor

The danger of *worldtext* is that it sounds like a graceful theoretical upgrade. It should sound more like a door opening in the floor. An image model gives a surface; a world model gives a place where surfaces answer back.

A picture can lie by resemblance; a world lies by consistency. The image says *“here is what appears”*; the world model says *“here is what can happen next.”* Time enters quietly; then the picture is no longer a picture, and the object is no longer finished by being seen.

We cannot keep calling this "image generation." The image is already becoming too small a category. The prompt is moving past the still image, past the video clip, past the illustration, past the surface. The prompt is becoming a way to summon navigable space, interactive environments, synthetic places, malleable scenes, and generated worlds.

The machine does not fulfill the ancient ekphrastic hope; it changes the scale of the hope. Immediacy was the old goal. Inhabitation is the new goal. Both promises are theological in disguise. Once the image can be entered, the old question returns in a harsher, materially grounded form: what kind of life has the computational description authorized?

The field has to move.

The field has to move from product to process.

The field has to move from imagetext to worldtext.

The field has to move from ekphrasis as description to ekphrasis as operation.

Humanists must stop treating generative AI as a vulgar technical novelty or a degraded image factory. They must claim prompting as a serious object of theory, a serious practice of craft, and a serious continuation of ekphrasis. They must treat generative AI not as an external threat to literary and visual theory, but as the machine that forces those theories to reveal what they were already trying to say.

The word wanted the image.

The image wanted the world.

The world wanted a procedure for being called forth.

Now, that procedure has a platform. The prompt is not the artwork; it is the initialization vector. The image is not the harvest; it is the first diagnostic denoise. The world model is not a bigger picture; it is the greenhouse becoming walkable. Human beings once asked language to preserve the world; now they ask language to compute one. The wish has changed less than we think.

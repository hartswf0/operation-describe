The Ekphrastic Object: From Image Output to World Model

The Obsolescence of the Imagetext

The old ekphrastic question asked whether words could make readers see. Generative AI appears to answer yes: the word becomes the picture. However, that answer is already obsolete. The decisive shift is not from word to image, but from word to world model. Prompting is becoming a cumulative, procedural, and increasingly spatial practice in which description no longer points at a world from the outside. Description helps constitute the world's operating conditions.

For the better part of the twentieth century, the dominant theoretical framework for ekphrasis was constrained to the "verbal representation of visual representation." This formalist boundary concealed the original operativity of the ancient rhetorical practice. Today, AI systems based on stochastic machine learning treat the text/image collapse not as a metaphor, but as a structural reality. Yet, this operative turn marks a threshold, not a destination. The image is already becoming too small a category. The prompt is moving past the still image, past the video clip, past the surface. It is becoming a way to summon navigable space, interactive environments, and generated worlds.

A prompt is not yet a world, but it has stopped being a sentence. The old ekphrasis asked whether words could make us see; the new one asks whether words can make something behave. A picture can lie by resemblance; a world lies by consistency. A world is not a larger image; it is an image with rules, durations, permissions, and refusals. The image says "here is what appears," the world model says "here is what can happen next." Time enters quietly; then the picture is no longer a picture, and the object is no longer finished by being seen.

Prompting as Performance and Thick Description

Prompting is often mistaken for description because it begins as language. That mistake keeps the analysis trapped at the wrong scale. A prompt does not simply say what an image should contain. A prompt tests what a model can make available, what a platform will default toward, what a system will resist, and what a user must restate, thicken, or redirect.

The old mistake is to imagine the prompt as a sentence that tells the machine what to draw. That is too thin. It treats generative AI like a vending machine: insert words, receive image. But anyone who has actually used these systems knows the first result is rarely the desired result. The first prompt does not finish the work. It starts the work.

A user writes a prompt, receives an output, reads the system's misinterpretation, revises the prompt, adds constraints, changes style, supplies reference images, removes noise, and tries again. The image is not the end of this process. The image is feedback. It tells the user what the model heard, what the model ignored, and what the model already assumes.

This is why "thick prompting" matters. The term should not become a decorative analogy to Clifford Geertz's anthropological methods; it should name the practical condition of working inside generative systems. Thick prompting is necessary because the model is already thick. Its outputs are shaped by captioned images, platform defaults, scraped styles, interface constraints, safety systems, hidden system prompts, training distributions, and probabilistic interpolation. A prompt never touches an empty machine. It enters a system packed with cultural residue.

Therefore, thick prompting does not mean writing longer prompts. A long prompt can still be thin if it merely piles up adjectives. Thick prompting means building a more explicit operating context. The user supplies not only subject matter but relations, constraints, exclusions, styles, temporal cues, spatial cues, and procedural instructions. In this sense, prompting moves closer to performance than to description. The prompt is performed against the system. The output answers back. The next prompt incorporates that answer.

The prompt becomes less like a caption and more like a score: a set of instructions that can be replayed, revised, interpreted, and varied by a system. It probes a model. It exposes the distance between what the user says and what the system can operationalize. There is no final perfect prompt because there is no final exhaustion of the model. The user never reaches the final prompt; the user only reaches the next useful version. Thick prompting names this unfinished condition. It is ekphrasis after the image has become feedback.

The Shield of Achilles as Worldtext

That is why the shield of Achilles matters. The shield is not the first great ekphrasis. It is a miniature world model. Homer's description does not only render an object; it organizes a cosmos of war, labor, law, agriculture, dance, weather, sea, and divine making. The shield is the hinge between operative text and generated reality: a described artifact that already behaves like a world-bearing system.

The shield of Achilles occupies a strange position in the Iliad: it is both inside the story and larger than the story that contains it. The Trojan War is the large narrative frame of the epic, but the shield reverses the scale. The war becomes one event inside a larger order. Macrocosm and microcosm exchange places. The shield is small enough to be carried by one warrior, but large enough to contain the pattern of existence that gives that warrior's violence its meaning.

Homer's shield matters because it already knew the trick: put a world onto a surface, then make that surface carry more life than its size should allow. Hephaestus makes the shield scene by scene. Homer makes the shield line by line. The poem does not freeze an image; the poem performs a process. The scenes on the shield imply movement. The wedding procession moves. The dancers turn. The armies clash. The ocean surrounds. It is a compressed environment.

This is where we propose the term worldtext. Worldtext does not simply mean "text about a world." Worldtext names a textual operation through which a world becomes available as a coherent field of relations. In an imagetext, the prompt and image form a verbal-visual compound. In a worldtext, the prompt, model, interface, and environment form a verbal-spatial-temporal compound. The word no longer becomes only picture. The word begins to condition world-state.

The Constitution of Generative Environments

Current world-model platforms do not yet fulfill the fantasy of complete verbal world creation. The genie is out of the bottle, but the world is not yet stable. Generated worlds remain partial, inconsistent, platform-bound, and heavily dependent on hidden scaffolding. The apparent magic of "word to world" is supported by an immense technical and symbolic infrastructure.

That infrastructure is exactly why worldtext matters. The prompt is not sovereign. The prompt is one operator among many. It collaborates with system prompts, training data, latent visual conventions, safety filters, physics engines, interface affordances, and procedural rules. A worldtext is not a single beautiful sentence; it comes from a stack of instructions that make the world cohere.

When we say "make a Homeric world," we do not ask for bronze, gods, ships, and ekphrastic dust; we ask for a grammar of possible action. A generated world needs more than appearance; it needs consequence. Without consequence, worldtext is only scenery with ambition. The word becomes picture: the old dream of the natural sign. The word becomes world: the newer and more dangerous dream. Immediacy becomes inhabitation, and both promises are theological in disguise.

Conclusion: The Burden on Humanists

The image is not a natural sign. The image is not pure vision. The image is not the thing itself finally rescued from language. The image is an image of images. It is a statistical memory of images, a platform's learned compromise between the prompt, the archive, the model, and the visual habits of a culture already trained to recognize itself.

Generative AI does not merely give ekphrasis a new tool. It exposes the operating logic that was hidden inside ekphrasis all along. The word wanted the image. The image wanted the world. The world wanted a procedure for being called forth. Now that procedure has a platform.

And that is where the second failure begins. We cannot keep calling this image generation. The imagetext is becoming worldtext. The burden is not to celebrate the machine. Not to denounce the machine. Not to polish old theory until it looks current. The burden is to rebuild the theory. Ekphrasis can no longer be treated as a minor literary device for describing art. Prompting can no longer be treated as a shallow user command. AI images can no longer be treated as finished pictures detached from the process that made them.

Humanists must claim prompting as a serious object of theory, a serious practice of craft, and a serious continuation of ekphrasis. They must stop treating generative AI as a vulgar technical novelty or a degraded image factory. They must rebuild ekphrasis around prompting, process, platform realism, and worldtext. They must treat generative AI not as an external threat to literary and visual theory, but as the machine that forces those theories to reveal what they were already trying to say: the word has always wanted to make an image, and the image has always wanted to become a world.
